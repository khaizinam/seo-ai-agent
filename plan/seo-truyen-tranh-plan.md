## 🚀 PHẦN 4: YẾU TỐ KỸ THUẬT SỐNG CÒN 2026 KHI SEO WEB TRUYỆN

Kinh nghiệm SEO website cấu trúc quá nhiều Img/Chap (đây là mỏ vàng nhưng cũng là tử huyệt):

1. **Hiệu suất hình ảnh (Core Web Vitals - Quan Trọng Nhất):**
   - Web truyện tranh **TUYỆT ĐỐI** dùng định dạng ảnh `WebP` (nén 70% so với JPG).
   - **BẮT BUỘC** có thẻ `loading="lazy"` cho mọi ảnh truyện dưới nếp gấp (below-the-fold), tránh tải cả chap 100 ảnh 1 lúc làm Server bạn chết và GG đánh rớt hạng vì tải chậm.
2. **Cấu trúc URL Chuẩn SEO (URL Hierarchy):**
   - Rất tốt: `/manhua/[slug-ten-truyen]/chap-123`
   - Bị GG phạt: `/doc-truyen?id=xxx&chap=123`
3. **Schema Markup Chuyên Sâu:** Sử dụng Schema JSON-LD `ComicStory` và `Series` ở trang chi tiết truyện để Google hiển thị Rich Results (Bao gồm Rate số sao đánh giá).
4. **Content Chống Canonical:** Các web auto-leech (cào dữ liệu) truyện rất đông! Hãy nhúng Watermark (Logo mờ `Khaizinam`) vào chính giữa 1 số khung tranh gốc, code chặn chuột phải chống Clone.
5. **Cộng đồng Kéo Time On-site:** Tích hợp form Comment (NuxtJS / Livewire) cực nhạy cuối mỗi Chap. Nếu User ở lại cãi nhau/hóng chap gõ cmt 3 phút ở trang của bạn -> Google lập tức đẩy web bạn lên top #1 vì **Dwell Time** cực khủng.
