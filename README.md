**SEOGEN AI** là một ứng dụng desktop mạnh mẽ được thiết kế để tự động hóa quy trình sản xuất nội dung chuẩn SEO, quản lý chiến dịch nội dung và tối ưu hóa sự hiện diện trên mạng xã hội. Với sức mạnh từ các mô hình AI tiên tiến nhất hiện nay, SEOGEN giúp bạn tăng tốc độ viết bài gấp 10 lần trong khi vẫn đảm bảo chất lượng và tính cá nhân hóa cao.

---

## 🚀 Tính năng nổi bật

### 1. Quy trình AI toàn diện (Full Process)
Chỉ với một cú click, SEOGEN thực hiện toàn bộ quy trình:
- **Sinh bài viết (1000-1500 từ)**: Định dạng HTML nén, chuẩn cấu trúc H2-H6, tối ưu mật độ từ khóa.
- **Tạo nội dung mạng xã hội**: Tự động viết bài quảng bá cho Facebook và LinkedIn với icon và hashtag phù hợp.
- **Tạo Prompt Thumbnail**: Tạo prompt tiếng Anh chuyên sâu cho các AI tạo ảnh (Midjourney, DALL-E) dựa trên nội dung bài viết.

### 2. Quản lý Nhân vật (Persona Management)
- Tạo và lưu trữ nhiều phong cách viết bài khác nhau.
- Tùy chỉnh giọng văn, phong cách và cách tiếp cận cho từng đối tượng độc giả khác nhau.

### 3. Lập kế hoạch & Chiến dịch (Campaigns)
- Quản lý danh sách từ khóa theo từng chiến dịch.
- Tự động hóa việc tạo bài viết theo lịch trình dự kiến.

### 4. Công cụ hỗ trợ tối ưu
- **Meta SEO Manager**: Quản lý và tối ưu hàng loạt Meta Title/Description.
- **SEO Audit**: Phân tích và chấm điểm bài viết theo các tiêu chuẩn SEO kỹ thuật.
- **Thumbnail AI**: Kho quản lý prompt và hỗ trợ sinh ý tưởng hình ảnh.
- **Image Converter**: Chuyển đổi định dạng ảnh nhanh chóng ngay trong app.

## 💻 Hướng dẫn Cài đặt & Phát triển

### Yêu cầu hệ thống
- **Node.js**: Phiên bản 18.x hoặc mới hơn.
- **Yarn**: Trình quản lý gói.

### Các bước cài đặt
1. **Clone repository**:
   ```bash
   git clone https://github.com/khaizinam/seo-ai-agent.git
   cd seo-ai-agent/seogen-ai
   ```

2. **Cài đặt dependencies**:
   ```bash
   yarn install
   ```

3. **Khởi chạy chế độ phát triển (Dev)**:
   ```bash
   yarn dev
   ```

### Xây dựng ứng dụng (Build)
Để tạo bản cài đặt `.exe` cho Windows:
```bash
yarn build:win
```
Bản cài đặt sẽ được tạo tại thư mục `dist-release/`.
